# Rock Paper Scissors Game 🎮

This is a simple Rock Paper Scissors game built using PHP.

## Features
- User vs Computer gameplay
- Random computer choice
- Emoji UI 🎮
- Simple and clean design

## How to Run
1. Install XAMPP or any local server
2. Place project in htdocs folder
3. Start Apache
4. Open in browser: http://localhost/rock-paper-scissors

## Technologies Used
- PHP
- HTML
- CSS
